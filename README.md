# Aneira Game

Aneira Game is a mobile-first educational mini-game app for young children.

This repository currently contains the Phase 1 foundation:

- Vite + TypeScript setup
- Phaser 3 bootstrapping
- shared app scenes
- starter game module structure
- age-group and game-catalog seed data

## Scripts

- `bun run dev`
- `bun run build`
- `bun run preview`

## Current Scope

- app shell foundation only
- first age group: 3 years old
- first game module: Animal Food
- no real gameplay yet
