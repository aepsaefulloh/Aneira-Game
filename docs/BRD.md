# BRD — Business Requirement Document

> This document follows `docs/WRITING_GUIDELINES.md`.

## Product Name

Aneira Game

## Product Summary / Ringkasan Produk

### Indonesia

Aneira Game adalah kumpulan mini-game edukatif sederhana untuk anak, dikelompokkan berdasarkan kategori umur.

Rilis pertama fokus pada anak usia 3 tahun dengan mini-game pertama bernama Animal Food.

### English

Aneira Game is a collection of simple educational mini-games for children, grouped by age category.

The first release focuses on children aged 3 years old with the first mini-game called Animal Food.

## Goals / Tujuan

### Indonesia

- Menyediakan game edukatif yang aman dan sederhana untuk anak kecil.
- Mengelompokkan game berdasarkan klasifikasi umur.
- Memulai dari MVP kecil yang sudah bisa dimainkan.
- Membangun fondasi yang dapat menampung lebih banyak mini-game.
- Menyiapkan project untuk rilis APK Android di masa depan.

### English

- Provide safe and simple educational games for young children.
- Organize games by age classification.
- Start with a small playable MVP.
- Build a foundation that can support more mini-games later.
- Prepare the project for a future Android APK release.

## Target Users / Pengguna Sasaran

- Primary: children aged 3–5 years old (initial focus: 3 years old).
- Secondary: parents or guardians who select the game category.

## Initial Target / Target Awal

- Age group: 3 years old.
- First game: Animal Food.
- Core learning: animal recognition, food recognition, object matching, hand-eye coordination, simple decision making.

## MVP Scope / Cakupan MVP

- Home screen
- Age selection
- Age 3 game list
- Animal Food mini-game
- Result screen

## Out of Scope for MVP / Di Luar Cakupan MVP

- login
- database
- online progress sync
- payment
- ads
- leaderboard
- complex parent dashboard

## Success Criteria / Kriteria Keberhasilan

### Indonesia

Anak dapat membuka Aneira Game, memilih Age 3, membuka Animal Food, menyeret makanan yang benar ke animal, menerima feedback positif, dan melanjutkan hingga result screen.

### English

A child can open Aneira Game, select Age 3, open Animal Food, drag the correct food to the animal, receive positive feedback, and continue through to the result screen.
