# Technical Decisions

## TD-001: TypeScript

Use TypeScript to keep scene, config, and content data safer as the project grows.

## TD-002: Vite

Use Vite as the web app toolchain for fast iteration and a lightweight setup.

## TD-003: Phaser 3

Use Phaser 3 for scenes, drag-and-drop interactions, animation, and audio handling.

## TD-004: No Backend for MVP

Do not introduce backend services during the MVP foundation and first mini-game phases.

## TD-005: Capacitor Later

Prepare the codebase so it can be wrapped later for Android, but do not add Capacitor in Phase 1.
