# 1.0 — Project Foundation

## Objective

Create the initial foundation for Aneira Game as a scalable educational mini-game platform for children.

This phase does not focus on full gameplay yet.
The goal is to prepare the project structure, documentation, app direction, and technical foundation.

## Product Context

Aneira Game is a collection of simple educational games grouped by age.

Initial scope:

* app name: Aneira Game
* first age group: 3 years old
* first mini-game: Animal Food
* first mechanic: drag and drop food to matching animal

## Technical Stack

Use:

* TypeScript
* Vite
* Phaser 3

Later:

* Capacitor for Android APK packaging

This phase is complete when:

* project can run locally with Vite
