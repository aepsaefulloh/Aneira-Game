# 1.1 — App Shell + Age Classification

## Status

Complete

## Phase Type

Foundation / App Shell

## Objective

Build the initial application shell for **Aneira Game** with age-based game classification.

This phase focuses on the main navigation flow, age category structure, and game catalog structure.

This phase **does not implement the full Animal Food gameplay yet**.

---

## Project Context

**Aneira Game** is a children’s educational mini-game platform.

The app will contain multiple simple educational games grouped by age classification.

Initial product direction:

* App name: **Aneira Game**
* First age group: **3 years old**
* First mini-game: **Animal Food**
* First mechanic: **drag and drop food to the correct animal**
* Future age groups: **4 years old** and **5 years old**
* Future mini-games should be easy to add through data configuration

---

## Current Scope

This phase should create the basic app flow:

```txt
HomeScene
↓
AgeSelectScene
↓
GameListScene
↓
AnimalFoodScene placeholder
↓
ResultScene placeholder
```

The user should be able to:

1. Open the app from `HomeScene`
2. Tap the Start button
3. Choose an age group
4. See available games for the selected age group
5. Tap `Animal Food`
6. Open the `AnimalFoodScene` placeholder
7. Navigate to a simple result screen placeholder

---

This app is intended for young children.

The UI should be:

* simple
* friendly
* colorful
* soft
* touch-friendly
* mobile portrait first
* easy for parents to navigate
* not visually crowded

* text must be short

* no timer
* no punishment

```ts
this.scene.start(SCENE_KEYS.GAME_LIST, {
```

Purpose:

Show available games for the selected age group.

For Age 3, show:

```txt
Age 3 Games

[Animal Food] Available
```

Behavior:

* Filter game catalog by selected `ageGroupId`
* Animal Food is clickable
* Coming Soon games are disabled
* Clicking Animal Food navigates to `AnimalFoodScene`
* Include Back button to return to `AgeSelectScene`

Example:

```ts
this.scene.start(game.sceneKey)
```

Reusable large button component.

* rounded rectangle background
* centered text
* pointer/touch interaction
* pressed visual feedback

```ts

  width: number
  height: number
```
