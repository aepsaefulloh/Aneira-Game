## Status

Done

## Phase Type

Documentation cleanup, UI consistency, and visual direction alignment.

## Objective / Tujuan

### Indonesia

Merapikan dokumentasi utama dan menyelaraskan UI agar pemilihan bahasa, sound toggle, dan tema setiap segmen umur lebih konsisten, proporsional, dan ramah anak.

### English

Clean up the main documentation and align the UI so language selection, sound toggle, and age segment themes feel more consistent, proportional, and child-friendly.

## Scope / Cakupan

### Indonesia

- Merapikan dokumentasi utama (terutama `docs/BRD.md` yang sebelumnya menumpuk semua dokumen).
- Memusatkan aturan bilingual ke `docs/WRITING_GUIDELINES.md`.
- Memperjelas arah visual di `docs/DESIGN.md`.
- Memperjelas aturan gameplay di `docs/GAME_DESIGN.md`.
- Menambahkan konsep tema berdasarkan age segment (`src/core/theme/age-themes.ts`).
- Merapikan UI language selector (`LanguageSelector`).
- Merapikan UI sound toggle (`SoundToggle`).
- Menyelaraskan layout HomeScene, AgeSelectScene, GameListScene, AnimalFoodScene, dan ResultScene.

### English

- Clean up main documentation (especially `docs/BRD.md`, which previously stacked every doc together).
- Centralize bilingual rules in `docs/WRITING_GUIDELINES.md`.
- Clarify visual direction in `docs/DESIGN.md`.
- Clarify gameplay rules in `docs/GAME_DESIGN.md`.
- Add an age segment theme concept (`src/core/theme/age-themes.ts`).
- Improve the language selector UI (`LanguageSelector`).
- Improve the sound toggle UI (`SoundToggle`).
- Align HomeScene, AgeSelectScene, GameListScene, AnimalFoodScene, and ResultScene layouts.
