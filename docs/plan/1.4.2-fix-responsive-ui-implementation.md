# Part 2 — Fix Phaser Scale Configuration

Check the Phaser game config.

Look for:

```txt
src/game.config.ts
src/main.ts
```

or any current Phaser config file.

Make sure the game uses a responsive-friendly scale mode.

Recommended direction:

```ts
scale: {
  mode: Phaser.Scale.FIT,
  autoCenter: Phaser.Scale.CENTER_BOTH,
  width: 390,
  height: 844,
}
```

For this child-focused mobile-first game, use a portrait-first base size.

Recommended base canvas:

```txt
390 x 844
```

Important:

```txt
- The game should fit within browser viewport.
- Do not stretch content weirdly.
- Center the canvas.
- Keep mobile portrait as the design baseline.
```

If the project already uses a different scale strategy, improve it without breaking the game.

The current HomeScene still looks too plain and too small.

Redesign HomeScene using the existing Age 3 / Animal Friends visual direction.

## Visual Requirements

HomeScene should include:

```txt
- warm pastel background
- soft rounded main panel/card
- larger title
- readable subtitle
- polished Start button
- proportional language selector
- proportional sound toggle
- better spacing on tablet and desktop
```

For tablet/desktop, the content should not stay tiny.

Use a centered content container with adaptive max width.

Suggested values:

```txt
Mobile content width: width - 32
Tablet/Desktop max content width: 520–640
```

The visual panel should feel larger on tablet, not stuck at small mobile size.
