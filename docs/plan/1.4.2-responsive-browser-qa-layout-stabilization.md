## Technical Notes / Catatan Teknis

### Layout Helper

```ts
// src/core/layout/layout.ts
export const computeAnimalFoodLayout = (width: number, height: number): AnimalFoodLayout => { ... }
export const SAFE_X = 24;
export const SAFE_Y_BTM = 32;
```

At the reference resolution `390×844`, all computed values are mathematically identical to the original hardcoded constants — no visual regression.

### Desktop Container

```ts
// src/main.ts
document.body.style.display = "flex";
document.body.style.justifyContent = "center";
appRoot.style.maxWidth = "480px";
appRoot.style.overflow = "hidden";
```

This keeps the portrait canvas in a centered column on wide screens rather than stretching to fill the full viewport width.

### Playwright Tests

Viewport matrix tested:

| Viewport | Width | Height |
|----------|-------|--------|
| mobile-small | 360 | 640 |
| mobile-standard | 390 | 844 |
| mobile-large | 430 | 932 |
| tablet-portrait | 768 | 1024 |
| tablet-landscape | 1024 | 768 |
| desktop | 1366 | 768 |

Canvas clicks use runtime game-coordinate mapping:

```ts
async function clickAtGame(page, gx, gy) {
  const box = await page.locator("canvas").first().boundingBox();
  await page.mouse.click(
    box.x + (gx / 390) * box.width,
    box.y + (gy / 844) * box.height,
  );
}
```
